# codex-session-tui

This npm package ships prebuilt binaries for `codex-session-tui`.

## Install

```bash
npm i -g codex-session-tui
```

## Run

```bash
codex-session-tui
```

## Platforms

Prebuilt binaries are published for:

- Linux: `x86_64`, `aarch64`, `armv7`
- macOS: `x86_64`, `aarch64`
- Windows (MSVC): `x86_64`, `aarch64`
