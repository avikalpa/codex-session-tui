# codex-session-tui

This npm package ships prebuilt binaries for `codex-session-tui`.

## Install

```bash
npm i -g codex-session-tui
```

## Run

```bash
codex-session-tui
```

The launcher automatically selects the binary for:

- Linux: `x64`, `arm64`, `arm` (armv7)
- macOS: `x64`, `arm64`
- Windows: `x64`, `arm64`
